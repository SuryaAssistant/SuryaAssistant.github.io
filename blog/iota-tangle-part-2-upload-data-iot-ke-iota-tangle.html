IOTA merupakan sebuah distributed ledger technology (DLT) yang dikembangkan oleh IOTA Foundation dengan berbasis di Jerman. IOTA memiliki tujuan yang sama dengan blockchain-blockchain pada umumnya, yaitu menciptakan media penyimpanan transaksi yang tidak bisa diubah (tamper-proof) dengan prinsip desentralisasi, namun menggunakan konsep yang berbeda.

Alih-alih menggunakan konsep block-chaining, IOTA lebih memilih menggunakan konsep directed acyclyc graph (DAG). DAG inilah yang kemudian dinamakan sebagai IOTA Tangle. Penggunaan DAG di IOTA menjadikan proses verifikasi tiap transaksi lebih cepat dibandingkan blockchain lain seperti Bitcoin maupun Ethereum yang memerlukan waktu hingga bermenit-menit untuk verifikasi satu block.



Directed Acyclic Graph. Sumber: Serguei Popov, 2019


Selain proses verifikasi yang lebih cepat, proses transaksi di IOTA tidak memerlukan biaya jaringan (fee) seperti pada Blockchain Bitcoin maupun gas fee seperti pada Blockchain Ethereum. Dengan demikian, proses penyimpanan banyak data kecil non transaksi seperti data Internet of Things (IoT) menjadi sangat memungkinkan.


Prequerities
Anda dianggap sudah memahami perintah-perintah dasar Ubuntu dan Bahasa Pemrograman Python sebelum mengikuti tutorial ini.

Tutorial ini dilakukan dalam lingkup "devnet" supaya tidak mengganggu jaringan "mainnet" IOTA.

1. Ubuntu

2. Python 3.6+

3. pip


Clone Repository

Buka terminal dan ketik command berikut untuk me-download Rust.

              
              $ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

dan pastikan sudah me-install libudev-dev.

              
              $ apt install libudev-dev

Buka terminal dan clone repository dari IOTA. 

              
              $ git clone -b production https://github.com/iotaledger/iota.rs


Build Library

Setelah me-clone repository, masuk ke dalam folder iota.rs/bindings/python/native.

              
              $ cd iota.rs/bindings/python/native

Lakukan instalasi dependency yang diperlukan dan tunggu hingga selesai.

              
              $ pip install -r requirements-dev.txt
              
            

              
              $ pip3 install maturin
              

              $ maturin build --manylinux off

Setelah proses selesai, pindah ke folder target/wheels untuk proses instalasi library. Nama file disesuaikan dengan nama file yg ada di folder target/wheels. Tiap komputer mungkin berbeda nama.

              
              $ cd target/wheels 
              
            

              
              $ pip3 install iota_client_python-0.2.0a3-cp36-abi3-linux_x86_64.whl

Dalam tutorial ini, data IoT yang digunakan berupa data dummy. Buat file python baru dan isikan kode di bawah ke file tersebut

              
                import iota_client

                client = iota_client.Client()


                # encoding utf string into list of bytes

                data_iot = "[{'suhu':32, 'kelembaban':60}]".encode("utf8")


                message = client.message(index="data_hari_ini", data=data_iot)


                print(message)

Jika berhasil akan muncul balasan berupa "message_id". Ini adalah alamat data yang sudah diupload ke dalam IOTA Tangle. Dalam kasus ini, "message_id" yang didapat adalah "0876ab1f1cca0cdfd420f895afb431036e53c491d13aa1ae6e3624377203a588"

Selain melalui IOTA Explorer, pencarian data juga dapat dilakukan melalui script python. Buat file python baru dan isikan kode berikut ke dalamnya. 

              
                import iota_client

                client = iota_client.Client()



                message = client.get_message_data("0876ab1f1cca0cdfd420f895afb431036e53c491d13aa1ae6e3624377203a588")

                message_payload = message["payload"]

                message_indexation = message_payload["indexation"][0]

                message_index = message_indexation["index"]

                message_data = message_indexation["data"]



                data_str=""


                index_str = bytes.fromhex(message_index).decode('utf-8')

                for i in message_data:

                  data_str += chr(i)



                print("Message index: ", index_str)

                print("Message data: ", data_str)
